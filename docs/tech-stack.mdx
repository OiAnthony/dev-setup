### Shell 别名配置

```bash
# 常用别名（配置在 ~/.zshrc）
alias docker="podman"              # 用 podman 替代 docker
alias code="code-insiders"         # 使用 VS Code Insiders
alias python="python3"
alias pip="pip3"
alias ll="ls -la"                  # 列出详细信息
```

### 快捷打开目录

```bash
code .          # 用 VS Code 打开当前目录
idea .          # 用 IntelliJ IDEA 打开当前目录
surf .          # 用 Windsurf 打开当前目录
zi <dir>        # 快速跳转到常用目录（zoxide）
```

## 使用偏好

基于命令历史分析，我的开发工作流偏好：

- **Agentic Coding 优先**：频繁使用 OpenCode、Claude Code、Windsurf 等 AI 辅助工具
- **多编辑器切换**：VS Code Insiders、IntelliJ IDEA、Windsurf 根据项目类型选择
- **Bun 开发体验**：使用 Bun 作为首选包管理器和运行时
- **Podman 容器化**：通过别名 `docker="podman"` 无缝使用 Podman
- **智能目录跳转**：使用 zoxide 的 `zi` 命令快速切换项目目录
- **API 调试**：结合 curl、openspec、Reqable 等工具进行接口测试

## 安装命令

```bash
# 安装开发工具
brew install node bun go rust python3 lazygit fzf fd starship zoxide htop btop uv

# 安装 oh-my-zsh
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

# 安装 SDKMAN!
curl -s "https://get.sdkman.io" | bash

# 安装 Volta
curl https://get.volta.sh | bash

# 安装图形应用
brew install --cask visual-studio-code-insiders iterm2 warp chat2db-pro redis-insight

# 环境变量配置
export PATH="/opt/homebrew/bin:$PATH"
eval "$(starship init zsh)"
eval "$(zoxide init zsh)"
source "$HOME/.sdkman/bin/sdkman-init.sh"
```
